들이와 무게

들이와 단위 알아보기

블록 1000개 만큼을 그릇에 담은 양
- 읽기 : 1리터
- 쓰기 : 1L
블록 1개 만큼을 그릇에 담은 양
- 읽기 : 1밀리리터
- 쓰기 : 1mL

1리터는 1000밀리리터와 같습니다.

단위가 있는 들이 비교하기

L단위부터 비교
2L < 3000mL < 4L
L가 같으면 m단위를 비교
5L 220mL < 5L 440mL

들이의 합과 차

- 들이의 합과 차를 구할 떄에는 L는 L끼리, mL는 mL끼리 단위를 맞추어 계산합니다.
-받아올림이 있는 들이의 합과 받아내림이 있는 들이의 차는 mL를 먼저 계산한 후, L를 계산합니다

무게의 단위 알아보기

물 1L의 무게
- 읽기 : 1킬로그램
- 쓰기 : 1kg
물 1mL의 무게
- 읽기 : 1그램
- 쓰기 : 1g

단위가 있는 무게 비교하기

kg 단위부터 비교
4kg 300g < 6kg
kg이 같으면 g단위를 비교
3kg 200g < 3kg 300g

무게의 합과 차

- 무게의 합과 차를 구할 떄에는 kg는 kg끼리, g는 g끼리 단위를 맞추어 계산합니다.
-받아올림이 있는 무게의 합과 받아내림이 있는 무게의 차는 g를 먼저 계산한 후, kg를 계산합니다
